# 04-验证测试

本目录用于按验证目的整理本项目使用的验证资产与说明入口。

当前只保留分类入口文档，便于后续继续补充验证脚本、结果索引和本地参考副本；大体量第三方测试树是否纳入版本控制，应单独评估后再提交。

## 当前分类

- `01-性能基准/`
  用于 CoreMark 等性能基准相关资料、说明和后续归档入口。
- `02-指令验证/`
  用于 `riscv-tests` 等指令功能验证相关资料、说明和后续归档入口。

## 使用约定

1. 验证结果的 authoritative summary 和日志仍以 `YH_rv_cpu/build/` 下实际产物为准。
2. 本目录更适合放验证分类说明、参考副本和整理后的辅助材料，不替代正式工程内脚本。
3. 如需纳入第三方基准或测试仓库，建议单独形成 focused commit，不与主线文档修改混提。
