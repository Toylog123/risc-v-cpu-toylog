# 集创赛工作区

## 一、项目概况

这个仓库当前只服务一个正式项目：

- 赛题：七星微《基于 `RISC-V` 的高性能 CPU 设计及 FPGA 验证》
- 正式工程：`toylog_cpu`
- 当前目标：先完成初赛闭环，再做性能优化和展示增强

## 二、当前做到哪一步

当前已经完成：

- 锁定七星微为唯一主赛题
- 建立正式工程 `toylog_cpu`
- 完成 `RV32I` 五级流水第一版
- 整理团队工具链说明与安装清单
- 重构 `01-项目管理`，补齐赛题要求、项目规划、工作记录、工作交接、任务清单

当前还没完成：

- `CSR / timer / trap`
- 最小 SoC
- `riscv-tests`
- `CoreMark`
- Vivado 工程与 FPGA 上板闭环

## 五、怎么协作

- 默认同步范围：
  - `toylog_cpu`
  - `04-工具链`
  - `01-项目管理`
- 默认暂存脚本：

```bat
toylog_cpu\scripts\stage_default_sync.bat
```

- 只看默认范围里有哪些改动：

```bat
toylog_cpu\scripts\stage_default_sync.bat --dry-run
```

- 每次对 `toylog_cpu` 有实际修改，都同步更新：
  - `toylog_cpu/doc/toylog_cpu_handoff.md`
  - `toylog_cpu/doc/toylog_cpu_change_log.md`
  - `toylog_cpu/doc/toylog_cpu_todo.md`
- 每次对项目安排、分工、交接、任务状态有实际修改，都同步更新：
  - `01-项目管理/03-过程管理/工作记录.md`
  - `01-项目管理/03-过程管理/工作交接.md`
  - `01-项目管理/03-过程管理/任务清单.md`
