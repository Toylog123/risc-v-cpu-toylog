## 使用说明

1. 正式源码与脚本主入口仍是 `YH_rv_cpu/`。
2. 赛题要求、提交材料和目录级说明统一从 `01-项目管理/` 进入。
3. 当前技术说明书的权威材料入口是：
   `01-项目管理/03-提交材料/技术说明书/`
4. 分赛区决赛材料准备从以下清单进入：
   `01-项目管理/03-提交材料/00-提交检查与清单/01-要求对照/分赛区决赛提交要求对照-2026-07-20.md`
5. `.worktrees/` 仅用于不同架构或性能探索；完整项目管理、参考资料、验证测试只维护在本主目录。
