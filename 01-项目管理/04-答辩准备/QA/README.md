# CICC1003618 初赛答辩 QA 准备说明

本目录用于准备评委问答环节。口径以已提交材料为准：技术说明书、性能与验证报告、PPT、源代码、FPGA 原型系统和上板视频。

## 使用顺序

1. 先看 `02-高频速答卡.md`，背熟 20 个最容易被问到的问题。
2. 再看 `01-评委QA完整题库.md`，按分类练习完整回答。
3. 最后看 `03-刁钻追问与防守口径.md`，专门练评委追问、质疑、挑战性问题。

## 现场回答原则

- 先给结论，再补证据，再说边界。
- 所有数据以冻结提交口径回答，不把探索路径说成提交能力。
- 不要宣称没有实现或没有验证的内容，例如不要说已实现 C 扩展。
- 遇到尖锐问题时，先承认边界，再说明取舍依据和后续方向。
- 不出现参赛单位、指导老师、个人身份等信息。
