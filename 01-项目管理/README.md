# 01-项目管理

本目录用于维护比赛项目的非源码正式材料。

## 子目录分工

- `01-赛题要求/`
  - 赛题原文、初赛要求、分赛区决赛要求、规则边界和评分口径
- `02-资料索引/`
  - 资料清单、资料链接索引和同步约定
- `03-提交材料/`
  - 按赛题交付类别组织的正式材料入口，包括技术说明书、源码包、FPGA 原型系统、性能与验证报告、演示视频和提交检查清单

## 当前重点入口

- 分赛区决赛要求：`01-赛题要求/分赛区决赛要求/第十届集创赛分赛区决赛比赛要求-整理版.md`
- 分赛区提交对照：`03-提交材料/00-提交检查与清单/01-要求对照/分赛区决赛提交要求对照-2026-07-20.md`
- 分赛区交接记录：`03-提交材料/00-提交检查与清单/02-冻结快照/分赛区决赛交接记录-2026-07-20.md`

## 使用建议

1. 赛题理解、交付核对和正式材料准备优先从这里进入。
2. 技术说明书当前权威材料入口是 `03-提交材料/技术说明书/`。
3. 分赛区决赛材料需基于最新代码冻结点重新生成技术数据、技术文档、PPT/PDF 和视频证据。
4. 目录或文件一旦改名，优先同步这里、根目录 `README.md` 和 `.codex-handoff.json`。
