# 01-项目管理

本目录用于围绕比赛项目维护“非源码但必须长期同步”的内容。

## 子目录分工

- `01-赛题要求/`
  赛题原文、答疑整理、规则边界和比赛口径。

- `02-执行管理/`
  项目总体规划、阶段计划、任务清单、工作记录、工作交接与历史归档。

- `03-资料索引/`
  资料清单、资料链接索引、Git 同步约定等“怎么找资料、怎么同步”的入口文档。

- `04-汇报与提交材料/`
  汇报摘要、阶段汇报、内部支撑材料、初赛真正提交材料和正式对外口径。

## 使用建议

1. 赛题理解、项目状态和交付材料优先从这里进入，不要只依赖工程目录内文档。
2. 目录或文件一旦改名，优先同步这里和根目录 README。
3. 过程记录与交接文档优先保留 dated 文件，方便追溯。
