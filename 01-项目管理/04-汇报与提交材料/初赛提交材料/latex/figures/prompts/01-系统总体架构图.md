# 图 1 系统总体架构图

## 最终提示词

```text
请生成一张正式技术说明书风格的工程化模块框图。
整体风格必须统一为：白底、扁平矢量、学术技术文档风格、深灰细线边框、浅蓝浅绿浅橙浅灰配色、结构严谨、模块对齐、留白充分、无装饰背景、无炫技效果。

文本规则必须严格遵守：
1. 所有模块名字使用英文。
2. 所有连接说明、路径说明、注解文字使用中文。
3. 不要生成长段文字说明。
4. 不要生成乱码，不要手写体，不要花哨字体。

请绘制一个小型 RISC-V SoC 的总体架构图，要求有清晰层级关系：
最外层边界框标题为 FPGA Top。
内部第二层边界框标题为 SoC。

SoC 内部包含以下模块：
左侧：CPU Core
右上：Instruction Memory
中间偏右：Address Decode
右侧中部：Data Memory
右下：MMIO Peripherals

模块之间的注解必须使用中文：
CPU Core 到 Instruction Memory：取指接口
CPU Core 到 Address Decode：访存与 MMIO 访问
Address Decode 到 Data Memory：数据存储访问
Address Decode 到 MMIO Peripherals：外设访问

整体布局要求规整、平衡、专业，像芯片设计文档中的系统框图。
不要加入透视，不要加入立体效果，不要加入电路板纹理，不要加入发光，不要加入海报化阴影。
模块内部保持干净，便于后续替换或细修。
```
