# 说明书配图 AI 提示词包

这个目录用于存放初赛设计说明书配图的 AI 出图提示词。

## 使用方式

1. 打开对应图号的 `.md` 文件。
2. 直接复制其中 `最终提示词` 整段去生成。
3. 如果平台支持负面提示词，把同一文件里的负面约束一起带上。
4. 生成图片后放回 `latex/figures/`，文件名保持不变。

## 统一要求

- 白底、扁平矢量、技术说明书风格。
- 使用深灰细线边框，浅蓝、浅绿、浅橙、浅灰作为辅助配色。
- 所有模块名只允许使用 prompt 中明确列出的英文标签。
- 所有箭头说明和注解只允许使用 prompt 中明确列出的中文短语。
- 严禁出现任何未列出的漂浮文字、乱码、重复标签、海报化装饰。
- 建议导出分辨率不低于 `2200x1400` 的 PNG。
