# 说明书配图 AI 提示词包

这个目录用于存放初赛设计说明书配图的 AI 出图提示词。

当前采用 `B` 方案：
- 每张图单独一个文件
- 每个文件都自带完整提示词
- 全部图使用同一套视觉语言
- 模块名字统一英文
- 注解和说明统一中文

## 使用方式

1. 打开对应图号的 `.md` 文件。
2. 直接复制其中 `最终提示词` 整段去生成。
3. 如果平台支持负面提示词，就把同一文件里的负面约束一起带上。
4. 生成图片后放回 `latex/figures/`。
