# 汇报与提交材料

本目录是当前比赛项目的正式汇报与提交材料主入口。

## 当前内容

- `汇报摘要.md`
  适合快速汇报、答辩或做 PPT 提纲时引用的压缩口径。

- `项目进展汇报-2026-04-01.md`
  阶段性进展汇报材料，可作为里程碑汇报底稿。

- `内部支撑材料/`
  项目内部维护的提交清单、冻结基线索引、答辩口径、说明书提纲和材料更新说明。

- `初赛提交材料/`
  当前真正面向初赛交付的目录，保留面向评审的正式设计说明书 PDF 与 LaTeX 源码。

## 当前维护规则

1. 正式材料优先维护在本目录，不再保留单独的旧汇报目录。
2. 对外真正提交的初赛材料只放进 `初赛提交材料/`。
3. 新的 fresh 结果若要升级为对外材料，必须先和 `YH_rv_cpu/` 主文档同步。
4. 初赛说明书默认保持“裁判视角正式版”写法，不回退到内部交接或材料组织口吻。
