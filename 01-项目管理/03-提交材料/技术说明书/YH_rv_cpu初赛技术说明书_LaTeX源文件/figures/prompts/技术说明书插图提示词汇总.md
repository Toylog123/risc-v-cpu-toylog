# 技术说明书插图提示词汇总

本文件用于统一保存技术说明书插图的生成提示词。所有插图统一要求为黑白灰配色、白色背景、正式技术说明书插图风格；每张图下方仅保留一段可直接用于生成图片的完整提示词。

## 图 1 系统总体架构图

黑白灰单色技术架构图，白色背景，正式 CPU 技术说明书插图风格，横向布局，画面中心是清晰规整的矩形框图系统，最外层为 “PYNQ-Z2 FPGA Prototype” 外框，内部主体为 “YH_rv_cpu SoC” 大框，SoC 左侧是 “CPU Core”，右侧依次排布 “Instruction Memory”“Address Decode / Bus Interconnect”“Data Memory”“MMIO Peripherals”，其中 “MMIO Peripherals” 内部再细分为 “UART TX”“Timer”“DONE / Status”，从 CPU Core 指向 Instruction Memory 标注 “instruction fetch”，从 CPU Core 指向总线标注 “load/store request”，总线再分别连接 Data Memory 和 MMIO Peripherals 并标注 “memory path”“mmio path”，从 Timer 回连 CPU Core 一条中断箭头并标注 “timer irq”，整体只使用黑、深灰、浅灰三色，线条均匀硬朗，留白充足，模块命名简洁统一，强调工程文档感，不要彩色，不要渐变，不要阴影，不要 3D，不要照片，不要海报风。

## 图 2 五级流水与关键控制通路图

黑白灰单色 CPU 五级流水技术框图，白色背景，正式硬件设计文档插图风格，横向布局，从左到右依次为 “IF”“ID”“EX”“MEM”“WB” 五个规整矩形模块，相邻模块之间明确标出 “IF/ID”“ID/EX”“EX/MEM”“MEM/WB” 流水寄存器，图下方单独放置 “Hazard Control” 控制模块，从 “EX/MEM” 和 “MEM/WB” 分别回连到 “EX” 并标注 “forward from EX/MEM”“forward from MEM/WB”，从 “EX” 回连到 “IF” 标注 “redirect”，从 “ID” 回连到 “IF” 画两条更强调的路径并标注 “ID branch early redirect”“ID JAL early redirect”，从 “Hazard Control” 指向 “IF/ID” 和 “ID/EX” 标注 “stall”“flush”，整体只使用黑、深灰、浅灰三色，模块规整，箭头清晰，层次紧凑，突出高性能控制流优化与前递、停顿、清空机制，不要彩色，不要渐变，不要阴影，不要装饰纹理，不要 UI 卡片风，不要宣传海报感。

## 图 3 验证闭环图

黑白灰单色验证闭环流程图，白色背景，正式芯片设计验证文档插图风格，采用自上而下三层结构：第一层为功能正确性验证，包含 “Directed Tests”“SoC Smoke”“RV32I Baseline”“Extension Tests”；第二层为性能与实现验证，包含 “CoreMark”“Dhrystone”“Vivado Synthesis”“Vivado Implementation”；第三层为硬件落地验证，包含 “Bitstream”“Hardware Download”“PYNQ-Z2 Bring-up”；最底部汇聚到 “Validation Closure” 和 “Evidence Archive”，右侧单独设置证据输出栏，列出 “summary”“waveform”“utilization report”“timing report”“program log”，并从相关验证阶段引出清晰箭头连接这些证据项，整体只使用黑、深灰、浅灰三色，矩形节点规整，层级分明，逻辑清楚，强调可追溯的工程验证闭环与证据链，不要彩色，不要渐变，不要霓虹效果，不要复杂图标，不要互联网产品流程图风格。

## 图 4 FPGA 原型适配路径图

黑白灰单色 FPGA 原型适配路径图，白色背景，正式硬件技术说明书插图风格，横向流程布局，左侧输入区包含 “RTL”“Firmware / Test Program”“Vivado Scripts”“Constraints”，中间处理区依次连接 “Synthesis”“Implementation”“Bitstream Generation”，右侧结果区包含 “Utilization”“Timing”“Bitstream”，最右侧或下方硬件验证区包含 “Vivado Hardware Manager”“PYNQ-Z2”“PROGRAM_OK”，并在结果区域附近以小型黑框标签标出关键指标 “50.0 MHz”“4934 LUT”“WNS +0.440 ns”，整体使用黑、深灰、浅灰三色，矩形模块规整，箭头方向统一，工程文档感强，突出从 RTL 到 bitstream 再到上板验证的完整落地路径，不要彩色，不要渐变，不要阴影，不要真实照片，不要立体透视，不要宣传海报风。
