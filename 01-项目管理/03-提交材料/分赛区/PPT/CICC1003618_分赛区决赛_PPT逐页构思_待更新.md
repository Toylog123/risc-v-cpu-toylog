# CICC1003618 分赛区决赛 PPT 逐页构思工作稿

本工作稿用于后续人工精修 PPT。当前分赛区目录已保留最新 PPT 源文件、PDF、31 页图片和纯图片版 PPT，主展示建议优先使用 PDF 或纯图片版。

## 叙事主线

评委应在 10 分钟内理解：本作品在资源受限 FPGA 场景下，以五级流水 RISC-V CPU 为核心，通过热点扩展、控制通路优化和完整验证闭环，形成可复核、可上板、可演示的提交成果。

## 页面组织建议

| 模块 | 目标 | 证据 |
|---|---|---|
| 核心预览 | 先给结果和复核入口 | 技术文档、技术数据包、PPT PDF |
| 架构设计 | 说明 CPU、SoC、FPGA 顶层职责 | 技术说明书、RTL 目录 |
| 流水控制 | 说明 stall、flush、forwarding 和 redirect | TestBench、波形与验证日志 |
| ISA 与性能 | 说明 Zmmul、Bitmanip 子扩展和热点收益 | CoreMark、Dhrystone 报告 |
| FPGA 上板 | 说明 PYNQ-Z2 运行链路和 UART 输出 | bitstream、Vivado 报告、UART 日志、视频 |
| 总结与复核 | 收束原创性、可行性、创新点 | 技术文档前置页和证据索引 |

## 制作要求

- 保持信息密度适合现场讲解，避免大段正文堆叠。
- 页面、备注和视频画面不得包含赛事禁用身份标识。
- 如继续改 PPT，需同步重新导出 `CICC1003618_分赛区决赛_PPT.pdf`、`image_final/` 和 `CICC1003618_分赛区决赛_PPT_纯图片版.pptx`。
