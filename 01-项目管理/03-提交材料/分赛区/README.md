## 上传前检查

1. 上传前优先选用 PDF 或纯图片版 PPT 作为主展示文件，PPT 源文件作为备用。
2. 上传后必须在线预览技术文档和答辩 PPT，检查是否能打开、内容是否完整、排版是否错乱。
3. 评分材料和比赛过程不得出现赛事禁用身份标识；海报为活动例外材料，不作为评分依据。
4. 若后续继续修改 PPT 或源码，需重新导出 PDF、逐页图片、纯图片 PPT、源码包、技术数据包和冻结哈希。
