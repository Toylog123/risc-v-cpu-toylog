## 当前主口径

- Candidate: `impl220_impl200_optExploreArea_routeAdvancedSkewModeling_postAggressive_cpu50`。
- 指标：9965 LUT / 6520 FF / 32 BRAM Tile / 8 DSP / 4.287521 CoreMark/MHz / 2.495618 DMIPS/MHz xsim / WNS +0.056 ns / WHS +0.121 ns。
- 边界：当前是 post-route timing-closed engineering candidate；PROGRAM_OK、board UART raw log 和同版视频未补齐前，不称 board-proven。

## 上传前检查

1. 优先上传 PDF 或纯图片版 PPT 作为主展示文件，PPT 源文件作为备用。
2. 上传后必须在线预览技术文档和答辩 PPT，检查能否打开、内容是否完整、排版是否错乱。
3. 评分材料和比赛过程不得出现学校名称、LOGO、简称、指导老师信息或成员真实姓名；海报为活动例外材料。
4. 若后续补齐 PROGRAM_OK/UART/video 或继续修改 RTL，需重新生成 PPT、技术文档、源码包、技术数据包和冻结哈希。
