# 程序运行环境说明

## 软硬件平台

- 主机系统：Windows + PowerShell
- 目标 FPGA 开发板：Xilinx PYNQ-Z2
- 目标器件：`xc7z020clg400-1`
- 板级 CPU 时钟：`50.0 MHz`

## 工具链

- RISC-V 交叉编译器：`riscv-none-elf-gcc` 或兼容的 `riscv32-unknown-elf-gcc`
- 仿真工具：Vivado `xsim`，部分缓存类测试可使用 `iverilog`
- FPGA 实现工具：Xilinx Vivado
- 脚本环境：Python 3、PowerShell、Windows Batch

## 主要复现步骤

1. 检查工具链：运行 `scripts/check_toolchain.bat`
2. 功能回归：运行 `scripts/run_soc_smoke.bat`、`scripts/run_zmmul_test.bat`、`scripts/run_bitmanip_test.bat`、`scripts/run_xthead_memidx_test.bat`
3. CoreMark 评测：运行 `scripts/run_coremark_score.bat`
4. Dhrystone 评测：运行 `scripts/run_dhrystone_score.bat`
5. FPGA 原型构建：运行 `scripts/build_pynq_z2_project.bat impl`

## 说明

- 默认脚本会自动解析常见工具安装路径；若本机路径不同，可在命令行中通过环境变量覆盖。
- 串口日志采集需要额外连接 `3.3V` USB-UART 模块到 Pmod B；PYNQ-Z2 的 Micro-USB 串口连接 PS 侧，不直接连接 PL 软核 UART。
- 若仅做仿真验证，无需连接开发板；若进行 bitstream 下载与板级验证，需要通过 PYNQ-Z2 的 Micro-USB/JTAG 链路连接主机。
