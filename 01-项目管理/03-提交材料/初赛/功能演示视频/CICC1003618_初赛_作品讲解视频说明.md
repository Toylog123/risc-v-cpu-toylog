# CICC1003618 初赛作品讲解视频说明

本目录用于放置面向评委的作品讲解视频，建议最终文件命名为：

`CICC1003618_初赛_作品讲解视频.mp4`

## 视频定位

作品讲解视频与 FPGA 原型系统上板演示视频分开管理：

- 作品讲解视频：用于讲清作品背景、CPU 架构、设计重点/难点/亮点、验证重难点和最终结果。
- FPGA 原型系统上板演示视频：已放入 `../FPGA原型系统/CICC1003618_初赛_FPGA原型系统上板演示视频.mp4`，用于展示 Vivado 下载、PYNQ-Z2 连线、CP2102 串口和 UART 实时输出。

## 建议内容结构

1. 作品概述：说明作品为基于 RISC-V 的五级流水 CPU 与 PYNQ-Z2 FPGA 原型系统。
2. 设计重点：RV32I 五级流水、Zmmul、Zba/Zbb/Zbs、流水线控制、低资源 FPGA 适配。
3. 设计难点：数据相关、控制相关、资源约束、50 MHz 时序闭合。
4. 设计亮点：`4.137461 CoreMark/MHz`、`2.908287 DMIPS/MHz`、`4961 LUT`、PYNQ-Z2 上板实时 UART 指标流。
5. 验证重难点和结果：指令/模块定向测试、SoC smoke、CoreMark、Dhrystone、Vivado 资源时序、PYNQ-Z2 下载和串口日志。
6. 总结：高性能、低资源、可复现、可上板；后续可整理为开源复现版本。

## 注意事项

- 视频中不得出现学校、学院、老师、导师或个人身份信息。
- 建议横屏录制，清晰度不低于 720p，优先 1080p。
- 若提交平台只允许一个演示视频入口，可将作品讲解视频与 FPGA 原型系统上板演示视频同时放入演示视频目录，或按平台允许规则上传到对应材料目录。
