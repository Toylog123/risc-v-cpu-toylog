# FPGA 原型系统提交说明

本目录用于保存 PYNQ-Z2 FPGA 原型系统相关提交材料。

## 当前文件

| 文件/目录 | 内容 |
|---|---|
| `fpga_artifacts_pynq_z2/` | bitstream、约束、Vivado 资源/时序报告、下载日志、UART 日志、连接图 |
| `CICC1003618_初赛_FPGA原型系统上板演示视频.mp4` | FPGA 原型系统上板演示视频，展示 Vivado 下载、PYNQ-Z2 连线、CP2102 串口和 UART 实时输出 |
| `CICC1003618_初赛_FPGA原型系统上板演示视频说明.md` | 上板演示视频说明和关联证据索引 |

## 与作品讲解视频的区别

FPGA 原型系统上板演示视频只证明板级下载、连接和实时运行证据。作品讲解视频单独放在 `../功能演示视频/` 目录，用于讲解作品设计重点、难点、亮点以及验证结果。
