# CICC1003618 初赛演示视频说明

本目录用于提交初赛演示视频。功能演示、性能验证和串口上板证据合并在同一个视频中展示，不再单独保留额外录制说明文件。

## 视频内容覆盖

| 赛题要求 | 视频中应体现的内容 |
|---|---|
| 作品讲解 | 介绍 `YH_rv_cpu` 的 RV32I 五级流水架构、SoC 组织、PYNQ-Z2 FPGA 原型系统 |
| 功能验证演示 | 展示仿真回归、关键波形、指令扩展测试和 Vivado 下载成功过程 |
| 性能验证演示 | 展示 `4.137461 CoreMark/MHz`、`2.908287 DMIPS/MHz`、资源和时序结果 |
| FPGA 上板证据 | 展示 Vivado Hardware Manager 识别 `xc7z020_1` 并出现 `PROGRAM_OK` |
| 串口输出证据 | 展示 `COM7 115200 8N1` 下连续输出 `YH_rv_cpu CoreMark/MHz=4.137461 DMIPS/MHz=2.908287 tick=XX pc=XXXXXXXX` |

## 上板连接口径

- 板卡：`Xilinx PYNQ-Z2`
- JTAG 下载：PYNQ-Z2 Micro-USB / Vivado Hardware Manager
- PL 侧 UART：Pmod B 外接 `3.3 V` CP2102 USB-UART
- 接线：CP2102 `RXD -> JB1 / Y14`，CP2102 `GND -> Pmod B GND`，CP2102 `VCC` 不连接
- 串口参数：`COM7`，`115200 8N1`，无校验，无流控

说明时应避免把 PYNQ-Z2 板载 Micro-USB 的 PS 侧串口误说成 PL 软核 UART。视频中看到的文本输出来自 Pmod B 引出的 PL 侧 UART 诊断输出，`tick` 连续变化，`pc` 为 CPU 调试 PC 采样。
