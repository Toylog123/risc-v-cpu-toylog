# YH_rv_cpu 汇报摘要

> 日期：2026-04-03

## 一句话总结

`YH_rv_cpu` 已形成可提交的比赛冻结基线，当前最重要的结果是 short CoreMark `0.912472`、strict CoreMark `0.912465` 且 `10.959325s`、`rv32 33/33`、`rv64 21/21`、`impl50 WNS = +5.822ns`，剩余事项已经收敛到实体板卡闭环和第二轮性能优化。

## 核心数据

| 指标 | 当前值 | 备注 |
|------|--------|------|
| 指令集 | RV32I + Zicsr，带 RV64 基线验证 | 当前主提交口径仍以 RV32 比赛流为主 |

| impl50 资源 | `2555 LUT / 2170 FF / 4 BRAM / 0 DSP` | fresh |
| impl50 时序 | `WNS = +5.822ns`，`WHS = +0.057ns` | fresh |
| FPGA-like probe | `156442 cycles`，`7.728811 CoreMark/MHz` | quick pre-board probe |

## 已完成

- 冻结并验证当前 CoreMark score 主线
- 补齐 strict EEMBC `>=10s` CoreMark 长跑证据
- 冻结并验证当前 riscv-tests rv32 / rv64 baseline
- 冻结并验证 `impl50` FPGA 比赛口径
- 完成 FPGA pre-board SOP、串口口径和证据归档入口
- 明确关闭“直接放松 `stall_decode` 继续提分”的高风险方向

## 剩余事项

- 实体板卡到位后的 UART/LED/boot banner 板级闭环

## 风险提示

- 当前 XDC 仍不是最终板级 I/O delay signoff 版本
