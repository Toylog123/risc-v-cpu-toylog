## RISC-V M扩展指令编码

M扩展指令使用opcode=0110011 (OP)，funct3/funct7区分具体指令：

| 指令 | funct3 | funct7[6:0] |
|------|---------|---------------|
| MUL | 000 | 0000001 |
| MULH | 001 | 0000001 |
| MULHU | 010 | 0000001 |
| MULHSU | 011 | 0000001 |
| DIV | 100 | 0000001 |
| DIVU | 101 | 0000001 |
| REM | 110 | 0000001 |
| REMU | 111 | 0000001 |

## 注意事项

1. 除法需要处理除零情况（RISC-V规定除零应产生全1）
2. 乘法结果溢出不需要特殊处理
3. 需要处理RV64的32位乘法指令（MULW, DIVW, DIVUW, REMW, REMUW）
