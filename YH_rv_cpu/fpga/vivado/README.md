# FPGA 上板说明

这个目录预留给 `YH_rv_cpu` 的 Vivado 工程。

## 当前状态

- 还没有最终 FPGA 顶层封装
- 还没有板级专用 `XDC`
- 还没有图形化模块连线设计

## 计划纳入的内容

- 板级专用顶层封装
- `XDC` 约束
- 存储器初始化流程
- `UART` 拉起
- `timer` 中断拉起
- 性能测量计数器

## 目标形态

第一阶段的 FPGA 目标建议遵循七星微题面方向：

- 采用 `Xilinx Zynq-7000` 或 `Artix-7` 级别开发板
- 板级稳定频率达到或超过 `50MHz`
- 先打通基于 `UART` 的演示路径
- 后续再接入 `CoreMark` 测试路径
