# 01-性能基准

本子目录用于整理性能基准相关资料与说明。

当前主线性能基准仍以 `YH_rv_cpu` 工程内 CoreMark 流程为准；如需在工作区内维护 `coremark/` 本地参考副本，应视为独立资产并单独评估是否纳入版本控制。

## 当前关注项

- CoreMark short / strict 口径说明
- CoreMark smoke 与分数复现链路
- profile 与优化实验的辅助材料索引

## 入口建议

- 正式说明：`YH_rv_cpu/doc/coremark_submission_report.md`
- 回归记录：`YH_rv_cpu/doc/regression_test_log.md`
- 性能实验记录：`YH_rv_cpu/doc/performance_experiment_log.md`
