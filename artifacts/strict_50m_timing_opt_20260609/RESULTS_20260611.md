## Important Validity Notes

- CoreMark algorithm sources were not modified.
- The CoreMark evidence is a short reproducible full-workload fast gate, not a strict EEMBC 10-second run.
- Matching bitstream is now archived for `impl60`. It was generated from the same routed checkpoint after the original timing run, and the post-checkpoint timing summary still reports `WNS +0.301ns / WHS +0.104ns` with all user timing constraints met.
- Board PROGRAM_OK, UART capture, and video evidence are still pending.
- DMIPS/MHz was measured in matching xsim Dhrystone evidence as `2.484735`; this is not board UART evidence.
- A lower-looking `4879 LUT / 0 DSP` implementation from 2026-06-11 is invalid for reporting because its FPGA extension generics did not match the CoreMark fast-gate ISA.
