# toylog_cpu 任务清单

## 使用规则

- 状态只用：`待办`、`进行中`、`已完成`、`阻塞`
- 有实际进展就同步更新，不等到阶段末尾
- 任务结论要和脚本、日志、文档对应

## 已完成

- `已完成` 建立 `toylog_cpu` 正式工程
- `已完成` 完成 `RV32I` 五级流水第一版
- `已完成` 补齐最小机器态 CSR / trap
- `已完成` 接入最小 machine timer interrupt
- `已完成` 建立交接、修改记录和任务清单机制
- `已完成` 统一说明文档和源码注释口径为中文
- `已完成` 打通 RISC-V 交叉编译链
- `已完成` 建立最小 SoC：
  - `ROM`
  - `RAM`
  - `UART`
  - `DONE`
  - `timer`
- `已完成` 打通 `xsim` SoC 烟测
- `已完成` 打通 `xsim` trap 烟测
- `已完成` 打通 `xsim` timer interrupt 烟测
