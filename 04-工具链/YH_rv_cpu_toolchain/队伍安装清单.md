# 七星微赛题队伍安装清单

这份清单不是“我这台机器装了什么”，而是给队友统一环境用的最低协作清单。目标是让大家都能：

- 拉取并阅读 `YH_rv_cpu`
- 跑 RTL 语法检查和基础仿真
- 编译裸机程序
- 继续做验证、上板和后续开发

## 必须安装

### 1. Git

- 用途：拉取仓库、提交修改、同步队友代码

### 2. Vivado 2025.2

- 用途：综合、实现、时序分析、比特流生成、调用 `xsim`

### 3. xsim

- 用途：模块级和系统级功能仿真
- 说明：当前队伍统一采用 `Vivado` 自带 `xsim`

### 4. Icarus Verilog

- 用途：本地快速语法检查
- 安装命令：

```powershell
scoop install iverilog
```

### 5. RISC-V 交叉编译器

- 当前统一命名：
  - `riscv-none-elf-gcc`
  - `riscv-none-elf-objdump`
  - `riscv-none-elf-objcopy`
- 推荐安装方式：

```powershell
npm install --location=global xpm@latest
xpm install --global @xpack-dev-tools/riscv-none-elf-gcc@15.2.0-1.1
```

## 建议安装

### 1. ripgrep

- 用途：快速搜代码和文档
- 安装命令：

```powershell
scoop install ripgrep
```

### 2. 串口工具

- 用途：后续 FPGA 上板看 UART 输出

## 可选安装

- `ModelSim / Questa`
- `VCS`
- `SymbiYosys / JasperGold`
- `Quartus`

这些都不是当前最低协作环境的硬门槛。

## 装完后的自检

先确认工具能找到：

```powershell
where.exe git
where.exe vivado
where.exe xsim
where.exe iverilog
where.exe riscv-none-elf-gcc
where.exe riscv-none-elf-objdump
where.exe riscv-none-elf-objcopy
```

再进入工程运行：

```powershell
cd <你的工作区路径>\YH_rv_cpu
cmd /c scripts\check_toolchain.bat
cmd /c scripts\check_syntax.bat
cmd /c scripts\build_firmware.bat
```

预期结果：

- `check_toolchain.bat` 能识别 `iverilog`、`xsim`、`Vivado`、`riscv-none-elf-*`
- `check_syntax.bat` 通过
- `build_firmware.bat` 成功生成 `build/sw` 下的固件文件
